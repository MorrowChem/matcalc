calc(structure: Structure | Atoms | dict[str, Any]) → dict[str, Any][source]ÔÉÅ

Performs the MD simulation calculation for the input structure. Prior to generating initial velocities, this method calls the superclass‚Äôs calc method for preprocessing and optionally relaxes the structure. It then initializes the atomic velocities using the Maxwell-Boltzmann distribution, runs the MD simulation, and returns the final energy along with the final atomic configuration.
